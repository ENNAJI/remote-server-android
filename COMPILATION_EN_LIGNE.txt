║          COMPILATION APK EN LIGNE - MÉTHODES GRATUITES                ║
╚════════════════════════════════════════════════════════════════════════╝

La compilation locale nécessite beaucoup de dépendances. Voici des méthodes
GRATUITES pour compiler l'APK en ligne sans rien installer!

┌────────────────────────────────────────────────────────────────────────┐
│ MÉTHODE 1: GITHUB CODESPACES (Recommandée - 100% Gratuit)            │
└────────────────────────────────────────────────────────────────────────┘

1. CRÉER UN COMPTE GITHUB (si vous n'en avez pas)
   - Allez sur: https://github.com/signup
   - Créez un compte gratuit

2. CRÉER UN NOUVEAU REPOSITORY
   - Cliquez sur "New repository"
   - Nom: "remote-server-android"
   - Public ou Private (au choix)
   - Créez le repo

3. UPLOADER LES FICHIERS
   - Cliquez sur "Add file" → "Upload files"
   - Glissez-déposez TOUT le contenu du dossier "Serveur Android APK"
   - Commit les changements

4. OUVRIR CODESPACES
   - Dans votre repo, cliquez sur "Code" (bouton vert)
   - Onglet "Codespaces"
   - Cliquez "Create codespace on main"
   - Attendez 1-2 minutes (un VS Code en ligne s'ouvre)

5. COMPILER L'APK
   Dans le terminal Codespaces (en bas), tapez:
   
   chmod +x gradlew
   ./gradlew assembleDebug

   Attendez 5-10 minutes la première fois.

6. TÉLÉCHARGER L'APK
   - L'APK sera dans: app/build/outputs/apk/debug/app-debug.apk
   - Clic droit sur le fichier → Download
   - Transférez sur votre Android!

┌────────────────────────────────────────────────────────────────────────┐
│ MÉTHODE 2: REPLIT (Très Simple - Gratuit)                            │
└────────────────────────────────────────────────────────────────────────┘

1. CRÉER UN COMPTE
   - Allez sur: https://replit.com/signup
   - Créez un compte gratuit

2. CRÉER UN NOUVEAU REPL
   - Cliquez "Create Repl"
   - Template: "Java"
   - Nom: "AndroidRemoteServer"

3. UPLOADER LES FICHIERS
   - Dans l'explorateur de fichiers (à gauche)
   - Uploadez tous les fichiers du projet

4. INSTALLER ANDROID SDK
   Dans le Shell, tapez:
   
   wget https://dl.google.com/android/repository/commandlinetools-linux-9477386_latest.zip
   unzip commandlinetools-linux-9477386_latest.zip
   
5. COMPILER
   chmod +x gradlew
   ./gradlew assembleDebug

6. TÉLÉCHARGER L'APK
   - Trouvez app/build/outputs/apk/debug/app-debug.apk
   - Téléchargez-le

┌────────────────────────────────────────────────────────────────────────┐
│ MÉTHODE 3: GITPOD (Automatique - Gratuit 50h/mois)                   │
└────────────────────────────────────────────────────────────────────────┘

1. CRÉER UN REPO GITHUB (voir Méthode 1, étapes 1-3)

2. OUVRIR DANS GITPOD
   - Allez sur: https://gitpod.io/
   - Connectez-vous avec GitHub
   - URL: gitpod.io/#https://github.com/VOTRE_USERNAME/remote-server-android
   - Un workspace s'ouvre automatiquement

3. COMPILER
   chmod +x gradlew
   ./gradlew assembleDebug

4. TÉLÉCHARGER
   - Téléchargez app/build/outputs/apk/debug/app-debug.apk

Pour une version ULTRA-SIMPLIFIÉE sans code:

┌────────────────────────────────────────────────────────────────────────┐
│ MÉTHODE 5: COMPILATION LOCALE SIMPLIFIÉE (Si vous insistez)          │
└────────────────────────────────────────────────────────────────────────┘

Si vous voulez vraiment compiler localement:

1. INSTALLER ANDROID STUDIO
   - Téléchargez: https://developer.android.com/studio
   - Installez (15-20 minutes)
   - Acceptez toutes les licences SDK

2. OUVRIR LE PROJET
   - File → Open
   - Sélectionnez "Serveur Android APK"
   - Attendez la sync Gradle (5-10 min)

3. COMPILER
   - Build → Build Bundle(s) / APK(s) → Build APK(s)
   - Attendez 2-5 minutes
   - L'APK sera dans: app/build/outputs/apk/debug/

┌────────────────────────────────────────────────────────────────────────┐
│ APRÈS COMPILATION - INSTALLATION SUR ANDROID                          │
└────────────────────────────────────────────────────────────────────────┘

Une fois l'APK compilé:

OPTION A: Via ADB (Téléphone connecté en USB)
  adb install app-debug.apk

OPTION B: Transfert Direct
  1. Copiez app-debug.apk sur votre Android (USB/Bluetooth/Cloud)
  2. Sur Android, ouvrez le fichier
  3. Paramètres → Sécurité → Autoriser sources inconnues
  4. Installez l'APK

OPTION C: Via Cloud
  1. Uploadez app-debug.apk sur Google Drive/Dropbox
  2. Téléchargez sur Android
  3. Installez

┌────────────────────────────────────────────────────────────────────────┐
│ DÉPANNAGE                                                              │
└────────────────────────────────────────────────────────────────────────┘

PROBLÈME: "SDK not found"
  → Utilisez GitHub Codespaces (SDK pré-installé)

PROBLÈME: "Gradle sync failed"
  → Vérifiez votre connexion Internet
  → Réessayez après quelques minutes

PROBLÈME: "Build failed"
  → Vérifiez que tous les fichiers sont uploadés
  → Essayez: ./gradlew clean puis ./gradlew assembleDebug

PROBLÈME: "Permission denied"
  → Tapez: chmod +x gradlew
